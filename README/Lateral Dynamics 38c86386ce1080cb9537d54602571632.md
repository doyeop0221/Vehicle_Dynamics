# Kinematic model

![image.png](Lateral%20Dynamics/463f94c1-7b20-4549-b1a4-5c16fa7d354a.png)

<aside>
💡

Kinematics 모델은 아래 사항을 전제한다.

- 각 휠의 속도 벡터는 휠의 방향과 같다. 즉, 각 바퀴에 Side slip이 일어나지 않는다.

이를 전제하면 다음과 같다.

$$
\dot x = vcos(\psi+\beta) \\
\dot y = vsin(\psi+\beta) \\
\dot \psi = {vcos\beta \over l_f + l_r } (tan\delta_f - tan\delta_r)
$$

각 기호는 다음을 의미한다.

- $\delta_f,\delta_r$ : 전,후륜 조향각
- $\psi$ : 차량이 바라보고 있는 방향 (자세) = Heading angle
- $\beta$ : Slip angle
</aside>

## Beta 구하기

![1000020492.jpg](Lateral%20Dynamics/c2bcf6c9-dbd0-456c-b4d3-99167e05dfff.png)

<aside>
💡

1. $\beta$는 차체의 Slip angle이다. 즉, $tan\beta = {V_y \over V_x}$이다.

2. 차량의 yaw rate는 전륜과 후륜에 각각 다르게 작용한다. 그림을 보면, 차량 질량중심을 기준으로, 후륜은 $V_y$와 반대방향으로, 전륜은 $V_y$와 같은 방향으로 작용한다.
휠 속도 벡터는 휠 방향과 같으므로, 다음과 같은 수식이 성립한다.
    
    $$
    tan\delta_f = {v_y - l_r * \dot \psi \over v_x} \\ tan \delta_r = {v_y + l_f * \dot \psi \over v_x}
    $$

# Dynamics model

## wheel sideslip angle

![image.png](Lateral%20Dynamics/d10d7ed2-c3e5-4c44-a8eb-c941dc35ec1f.png)

<aside>
💡

- Wheel Sideslip은 다음과 같다.

$$
\alpha_f = \beta + {l_f * r\over v_x} - \delta_f = \beta + {l_f * \dot \psi \over v_x } - \delta_f\\ \alpha_r = \beta - {l_r*r \over v_x } = \beta - {l_r * \dot \psi \over v_x}
$$

- 이때, Linear tire model을 적용하면, 다음과 같다.

$$
F_{yf} = -C_f \alpha_f = -C_f (\beta + {l_f * \dot \psi \over v_x } - \delta_f) \\ F_{yr} = -C_r \alpha_r = -C_f ( \beta - {l_r * \dot \psi \over v_x})
$$

이를 힘 평형과 모멘트 평형 식을 활용하면 다음과 같다.

$$
m(\ddot y + r*v_x) = F_{yf} + F_yr = -C_f (\beta + {l_f * \dot \psi \over v_x } - \delta_f) -C_f ( \beta - {l_r * \dot \psi \over v_x}) \\ \therefore \dot \beta = -r + {C_f\over mv_x}(-\beta - {l_f*r \over v_x}) + {C_r\over mv_x}(-\beta + {l_r*r\over v_x}) (\because \dot \beta = {\ddot y\over v_x}) \\= -\dot \psi + {C_f\over mv_x}(-\beta - {l_f*\dot \psi \over v_x}) + {C_r\over mv_x}(-\beta + {l_r*\dot\psi \over v_x}) \\ \dot r = \ddot \psi = {C_fl_f \over I_z}(-\beta - {l_f * r\over v_x}+\delta_f)-{C_rl_r \over I_z}(-\beta + {l_r * r\over v_x})\\ ={C_fl_f \over I_z}(-\beta - {l_f * \dot\psi \over v_x}+\delta_f)-{C_rl_r \over I_z}(-\beta + {l_r * \dot\psi \over v_x})
$$

즉, $\dot \beta, \dot r$을 적분해 사용!

- Simulink 구현
    
    ![image.png](Lateral%20Dynamics/image%202.png)
    

→ $v_x, v_y$를 구하고, 이를 적분하면 차량이 움직인 변위를 알 수 있음!

![image.png](Lateral%20Dynamics/image%203.png)

</aside>

## Dynamic tire model

<aside>
💡

Dynamic tire model은 다음과 같다.

이때,  
$\tau = {L\over v_x} (\because \tau = {C_\alpha \over K v_x} , L ={C_\alpha \over K}  )$, L은 relaxtation length.

$$
\tau \dot F_{y,lag} + F_{y_,lag} = F_y \\
\therefore \dot F_{y_lag} ={F_y - F_{y,lag}\over \tau} = {V_x \over L}*(F_y - F_{y,lag})
$$

즉, $\dot F_{y,lag}$를 구하고, 이를 적분해 피드백 루프를 돌리면 구현 가능.

Laplace transform은 다음과 같다.

$$
F_{yf,lag} = {F_{yf} \over \tau s+1} \\ F_{yr,lag} = {F_{yr}\over \tau s+1}
$$

## Understeer Gradient

<aside>
💡

- $K_v = ({ml_r\over C_fL}- {ml_f\over C_rL})$임을 활용해, Understeer gradient를 계산할 수 있다.

![image.png](Lateral%20Dynamics/image%207.png)

- $\delta_f = {L \over R} + K_v* {V^2_x\over R}$를 $K_v$에 대한 식으로 바꾸면, $K_v = {\delta_f*R -L\over v^2_x}$이다.
이때, 회전반경 $R = {v_x\over \dot \psi}$이므로, 이를 활용해 Understeer gradient를 구할 수 있다.
즉, 주행 중 측정한 데이터를 활용해 Understeer gradient를 계산할 수 있다.

![image.png](Lateral%20Dynamics/image%208.png)

- Understeer gradient가 양수면 Understeer, 0이면 Neutral steer, 음수면 Oversteer 성향을 띠는 차량이다.
</aside>

## Characteristic speed, Critical speed

<aside>
💡

- Characteristic speed : 조향각이 애커만 각의 두배가 될 때의 속도
- Critical speed : 조향각이 0이 될 때의 속도

정상상태(Steady-state) 선회에서, 차량이 Understeer 성향이면, Characteristic speed일 때 yaw 응답이 최대가 된다. 즉, Understeer 성향의 차량이 조향 입력에 가장 민감하게 반응하는 속도이다.
반면, Oversteer 성향의 차량이 Critical speed일 때, yaw 응답이 무한대로 발산한다. 즉, Oversteer 성향의 차량이 해당 속도에 도달하면, 아주 작은 조향 입력에도 Spin한다. → Unstable.

- Simulink 구현 결과
    - $v_{char} =\sqrt {L\over K_v}, v_{crit}= \sqrt {L\over -K_v}$
        
        ![image.png](Lateral%20Dynamics/image%209.png)
        
    - $\delta_f = {L \over R} + K_v * {v^2_x\over R}$을 활용하여 $\delta_f$를 구한 결과가 $2L \over R$과 일치하는 속도 = $v_{char}$, 
    0과 일치하는 속도 = $v_{crit}$
    
    ![image.png](Lateral%20Dynamics/image%2010.png)
    
</aside>

## Desired yaw rate

<aside>
💡

- desired yaw rate : 운전자가 조향각 $\delta_f$를 주었을 때, 차량이 정상상태에서 내야하는 이상적인 yaw rate

$$
r_{des} = \dot \psi _{des} ={v_x \over R} = {v_x\over 1+({v_x\over v_char})^2}{\delta_f \over L}
$$
